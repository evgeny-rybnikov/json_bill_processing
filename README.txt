    ==========================
    ==== HOME BILLING APP ====
    ==========================

    HOW TO USE:
    Place json file(s) to 'D:\bills\json'
    Run Main::main
    If "Unknown INN" exception has been thrown, add new INN to 'inn.properties' file
    Copy data from console and paste them to Excel
    Move json(s) to 'D:\bills\json\done' (or delete if no archive required)
